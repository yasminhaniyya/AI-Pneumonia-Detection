# Pneumonia Detection using YOLO

This project implements an object detection model using YOLO to identify pneumonia indications from chest X-ray images.

## Technologies
- Python
- YOLO (You Only Look Once)
- OpenCV
- NumPy

## Workflow
1. Data collection and preprocessing of chest X-ray images
2. Annotation of dataset for object detection
3. Model training using YOLO architecture
4. Model evaluation and testing on validation data

## Features
- Detects pneumonia indications from medical images
- Uses deep learning-based object detection model
- Supports prediction on new input images
